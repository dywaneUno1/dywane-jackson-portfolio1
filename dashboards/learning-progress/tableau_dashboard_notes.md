# Learning Progress Dashboard - Design Notes

## Dashboard Components

1. **KPIs**: Total hours, weeks completed, average weekly hours
2. **Weekly Trend**: Line chart showing study hours over time
3. **Confidence Progression**: Area chart of confidence levels
4. **Topic Distribution**: Breakdown of time by subject

## Data Source

- ProgressData.csv (updated weekly)
- Tracks Python, SQL, and DevOps learning journey
